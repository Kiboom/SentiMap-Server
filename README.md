# SentiMap Server Application
Node.js와 Express를 활용한 SentiMap Server Application입니다.  
[SentiMap iOS github](https://github.com/Kiboom/SentiMap)

## 사용 기술  
- Node.js
- Express.js
- MySQL

## 서비스 개요  
간단한 조작만으로 순간의 감정을 표현할 수 있고,  
지도 위에서 전세계인들과 
시각화된 감정을 실시간으로 공유 가능한 위치 기반 SNS
